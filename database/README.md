# Database Schema - Ektor Pool Services

This directory contains the MySQL database schema for the pool maintenance business.

## Files

- `01_schema.sql` - Table definitions and constraints
- `02_seed.sql` - Sample data for testing
- `03_queries.sql` - Example queries
- `ERD.png` - Entity Relationship Diagram

## Quick Setup
```bash
mysql -u root -p < 01_schema.sql
mysql -u root -p < 02_seed.sql
```

See main README for full documentation.
